## Installation

Prerequisites:
- [Node.js] 4+, **npm** 3+.
- [ESLint] 4+

```sh
npm install --save-dev eslint eslint-config-template
```

## Usage

Add this to your `.eslintrc.json`:
```json
{
  "extends": [
    "template"
  ]
}
```
